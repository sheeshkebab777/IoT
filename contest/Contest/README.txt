To choose which Node the device is, you have to press the upper right button (2nd Button).
Depending on what Node the device is set to (at the beginning Node 1 (sink node)), the
according LED will light up (Node 1, Led 1...).

To start sending, press the first button (upper left) on the sink node.
When the sending has started, all 4 LEDs will light up on the sink node.
Press the start sending button, until all 4 LEDs light up.
